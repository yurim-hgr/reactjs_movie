# MOVIEWEB

- react js를 활용한 영화 정보 웹사이트 프로젝트

- 웹사이트 들어가기 : https://yurim-hgr.github.io/reactjs_movie

### 사용스킬

- HTML / CSS
- Reactjs - useEffect, useState

### 주요 기능

1. API 를 통해서 가져온 정보를 화면에 리스트로 렌더링해준다.
2. 영화를 클릭하면 영화 상세정보로 화면이동한다.

### 보완점

- 상세화면 밑에서 조금 띄우기
- 한국영화로 만들어보기
- 배우 설명도 추가해보기
- 예고편나오게
- 홈화면에 슬라이드 만들기
